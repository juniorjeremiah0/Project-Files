About Me

My name is Jeremiah Junior, and I am currently working towards completing an Associate of Applied Science (AAS) in Computer Support. This degree is a step in my journey to solidify my foundational IT skills and build a rewarding career in technology.

Originally, I started as an intern in the IT field, learning on the job and gaining practical skills. Over time, I discovered a strong interest in audio-visual technology, which combines hands-on work with technical problem-solving. Now, in my professional role, I get to tackle technical challenges daily, which keeps me motivated and engaged.

In addition to my work, I’m also committed to continual learning. This class is one part of that commitment, helping me gain new skills and broaden my understanding of web development. Each lesson brings me one step closer to my goal of becoming a well-rounded IT professional.

My Learning Experience

Since beginning this course, I have been introduced to the basics of HTML, CSS, and web development practices. Learning to structure a webpage, organize content, and apply semantic tags has been invaluable. Understanding how to format content correctly ensures that webpages are accessible and easy to maintain.

Additionally, I’ve been practicing using inline elements like strong and emphasis tags to highlight important parts of text. These elements allow me to add subtle stylistic touches that enhance readability and engagement. Mastering these basics has been challenging but rewarding, and I feel more confident in my ability to build professional-looking web pages as a result.

Beyond HTML, I am also learning the fundamentals of version control using Git and GitHub, which are essential skills for any IT professional. Through Git, I can track changes in my code, collaborate more effectively, and manage multiple versions of my project. This practical knowledge will benefit me not only in web development but also in other IT areas where version control is crucial.
